# Task Manager

## Overview

The **Task Manager** is a simple and intuitive to-do list application designed to help you manage and organize your tasks efficiently. This application allows users to add, edit, delete, and view tasks. It features a clean user interface and provides basic task management functionalities to enhance productivity.

## Features

- Add new tasks with a title and description.
- Mark tasks as completed or pending.
- Edit or update existing tasks.
- Delete tasks that are no longer needed.
- View tasks in a list format.
